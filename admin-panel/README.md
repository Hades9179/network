# 🛡️ Сүлжээний Админ Самбар

Админ нь сүлжээний сайтуудыг үр ашигтай удирдах, хяналт шалгах, мониторинг хийх ажиллагаа нь төвлөрсөн.

## 🎯 Үндсэн Функцүүд

### 📊 Үндсэн Мэдээлэл
- 📈 Нийт сайтуудын тоо
- ✅ Идэвхтэй сайтууд
- ⚠️ Анхаарах сайтууд
- ❌ Идэвхгүй сайтууд
- 🔄 Сүүлийн үйлдлүүд

### 🌐 Сайтуудыг Удирдах
- ➕ **Сайт нэмэх** - Нэр, URL, тайлбар
- ✏️ **Сайт засах** - Мэдээллийг өөрчлөх
- 🗑️ **Сайт хасах** - Системээс устгах
- 🔍 **Статус шалгах** - Сайтын идэвхжил хэмжих
- 🔎 **Хайх** - Нэр эсвэл URL-ээр хайх
- 📋 **Сүлүүлэх** - Статусаар сүлүүлэх

### 📡 Сүлжээний Хяналт
- ▶️ **Хяналт эхлүүлэх** - Автоматаар мониторинг
- ⏹️ **Хяналт зогсоох** - Мониторингийг завсрууулах
- 📋 **Лог харах** - Хяналтын түүх
- 🗑️ **Лог цэвэрлэх** - Түүхийг арилгах

### 📈 Шинжилгээ
- 📊 Сайтуудын ажилласан цаг
- 📉 Идэвхилт статистик
- 🔄 Шалгалтын түүх

### ⚙️ Тохиргоо
- ⏱️ Хяналтын интервал (сек)
- 📧 Администраторын имэйл
- 🔔 Уведомлението URL

## 🚀 Ашиглалт

### 1️⃣ Нээх
- `admin-panel/index.html` файлыг браузерээс нээнэ

### 2️⃣ Сайт нэмэх
- **🌐 Сайтууд** табт очиж **+ Сайт нэмэх** товчыг дарна
- Сайтын нэр, URL, тайлбар оруулна
- **✅ Хадгалах** дарна

### 3️⃣ Статус шалгах
- Сайтын карт дээр **🔍** товчыг дарна
- Автоматаар сайтын статус шалгагдана

### 4️⃣ Мониторинг идэвхжүүлэх
- **📡 Хянаж байна** табт очиж **▶️ Хяналт эхлүүлэх** дарна
- Систем автоматаар сайтуудыг мониторинг хийх болно

### 5️⃣ Сайтыг засах эсвэл хасах
- Сайтын карт дээр **✏️** (засах) эсвэл **🗑️** (хасах) товчыг дарна

## 💾 Өгөгдөлийн Хадгалалт

Бүх мэдээлэл **Local Storage**-д автоматаар хадгалагдана:

```javascript
// Сайтуудын хадгалалт
localStorage.setItem('adminPanel', JSON.stringify(websites));

// Үйлдлийн түүх
localStorage.setItem('adminPanel_log', JSON.stringify(activityLog));

// Тохиргоо
localStorage.setItem('adminPanel_settings', JSON.stringify(settings));
```

## 🔍 Сайтын Оруулах Мэдээлэл

Нэмэгдсэн сайт доорх өгөгдөл агуулна:

```javascript
{
  id: 1234567890,                    // ID
  name: "Google",                    // Нэр
  url: "https://google.com",         // URL
  description: "Хайлтын систем",    // Тайлбар
  status: "online",                  // Статус (online/offline/warning)
  checkCount: 5,                     // Шалгалтын тоо
  uptime: 99.8,                      // Ажилласан хувь
  lastCheck: "12:30:45",             // Сүүлийн шалгалт
  createdAt: "5/15/2026 12:00:00"   // Үүсгэгдсэн цаг
}
```
